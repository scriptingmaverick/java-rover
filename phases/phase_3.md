# Phase 3: Multiple Rovers (No Collision)

**New Requirements**

- Multiple rovers
- Each rover has ID
- Instructions specify rover

```text
5 5
R1 1 2 N
R2 3 3 E
R1: FFRFF
R2: FFF
```

**Expected Behavior**

- Each rover executes independently
- No collision handling yet
- Expected output a collection of rovers state
